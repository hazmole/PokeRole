訓練家的基礎

獨自在荒野中旅行是相當辛苦的。你的寶可夢夥伴會保護你的安全，但牠們沒有辦法讓你過得更舒適。大多數時候，你會需要睡在岩石床上，並徒步行走數百公里的路途。

認真的訓練家會需要攜帶道具以幫助他們達成自己的目標，同時減輕他們在旅途中可能會遭遇的不便。

每個訓練家都應該隨時備有這些道具：

金錢

你會需要金錢來買東西。一名剛踏上旅途的訓練家通常會有 $1500 可以使用。你可以透過與其他訓練家的對戰獲勝來賺到更多的寶可幣，但你也可能會因為被打敗而失去它們！

你也可以透過找一份小兼差來賺到更多金錢。試著找找看那些會同時雇傭人類和寶可夢的工作，好讓你的夥伴們也能幫上你的忙。透過這個方式，你不僅可以賺到一些額外的收入，也能幫助到那些有需要的人們。

食物

你需要給你自己和你的寶可夢的食物。有些訓練家會讓他們的寶可夢自由閒晃，自己去找些吃的，但親自餵食牠們總是更安全一些。畢竟，有些寶可夢在其他飢餓的野生寶可夢眼中，看起來就跟美味的晚餐沒什麼兩樣。

寶可夢道具

你的寶可夢會保護你不受到危險，但牠們的安危就是你的責任了。你應該至少要準備一些治療道具好讓牠們恢復健康，以及精靈球以安全地帶著牠們移動。

露營裝備

在野外要找到安全的庇護所並不容易。那些討厭的生物往往會棲息在洞窟和藏身處，所以除非你想要睡在滿是超音蝠大便的洞穴地面上，否則你最好帶上帳篷、睡袋、或至少一些日常使用的工具。

交通工具

城鎮之間的距離可能相當漫長。訓練家們會盡可能加快他們旅行的步調，儘管最富有的那群人會使用汽車或摩托車代步，大部分的人們更偏好騎乘自行車，讓旅行速度更快一些，而不需要在燃料上多花錢。也有些訓練家會騎著他們的寶可夢四處旅行。但如果你沒有辦法做到上面這些方法，那你最好買雙舒服點的鞋子，老老實實的走路吧。

訓練家裝備

如果你有合適的工具，那麼露營可以是一件很有趣的事。以下是一些你旅途中可以攜帶的裝備的例子。你所購買的每個沉重或大型的道具都會附上一個它自己的精靈球，協助你減輕它的重量，並能快速地把它收起來。

寶可夢世界中的城市和城鎮就跟我們世界的一樣，有著販售各種商品和服務的的商店。你的玩家們可以以合適的價格購買各種道具。

但記得要強調他們使用寶可夢的招式和特性來克服他們面對的挑戰。比如說，與其花錢買繩子，為什麼不乾脆讓你的妙蛙種子使用藤鞭就好呢？

關於販售道具的快速筆記

如果你手頭困窘，你永遠可以試著把你身上的一些道具賣給友好商店。

記住，他們能提供給你最好的價格只會是該道具的市場售價的一半，他們總是要賺錢的嘛！
但如果你找到了他們通常不會販售的稀有道具，那他們會慷慨得多！

你也可以在城鎮中直接找到買家，他們可能會想要某些你能提供的道具。

旅途用的道具

正如前面提到的，在寶可夢的世界中，旅途距離可能會相當漫長的。你不只需要穿過道路，你還會需要穿越森林、山脈、雪山、沙漠、海洋、或甚至飛越蒼穹。你當然可以只憑你的雙腳和結實的鞋子就周遊世界，但有一些其他東西來幫助你旅遊也不是個壞主意。

如果你夠幸運的話，你甚至有機會捕捉到稀有的寶可夢、或找到埋藏在洞窟深處的寶藏！

防身道具

儘管大多數時候你會希望尋找並收服新的寶可夢，但有時候你的隊伍可能會過於疲勞以至於無法好好的保護你、或也許這個區域附近遊蕩著一些太危險的怪物，無論是哪種情況，保持安全總是好的。

確定你自己知道什麼時候該從野外的戰鬥中逃走。大部分的寶可夢並不會追出牠們的領地，但如果你覺得自己很勇敢，你最好準備好一些道具或優秀的隊伍來保護你不受傷害。拼命戰鬥到你所有的寶可夢倒下可不是個明智的主意，這會讓你直接陷入危險。

治療道具

在這個世界上有著種類繁多的治療道具。大部分基本的治療道具會在友好商店中被販售，而其他的則能在野外被發現──只要你知道該如何以及往哪裡去找。
如果你想要有一趟更安全的旅途，那麼你將會需要常常帶著它們。

傷藥

傷藥通常會以噴霧的形式呈現，它們是止痛劑跟一些其他治療藥物的混合，幾乎能在一瞬間治好傷口。你使用傷藥治療的每一點傷害都將消耗瓶子中 1 單位的傷藥；治療每一點致命傷害則會消耗 2 單位。

如果寶可夢陷入瀕死狀態，你可以對牠使用傷藥來穩定牠的傷勢，但牠並不會因此恢復意識。

膠囊形式的傷藥僅供一次使用，且其劑量濃度非常高，以至於它需要經過 24 小時才能從體內被排出，讓其他傷藥再次作用於身體上。

樹果

寶可夢會牠們覺得必要的時候吃掉牠們持有的樹果，通常這會發生在牠們的HP降到一半、或牠們開始感覺到異常狀態發揮作用。牠們會比預期更快的吃掉樹果、或在牠們訓練家的指示下這麼做。

中藥

儘管樹果相當美味，但不會永遠是它們結果的季節，而世界上的某些地方也可能根本找不到它們。在古代，人們求助於特殊的草藥和植物根莖來製作高品質的藥劑；然而由於這些藥劑的味道全都像不良蛙的毒囊一樣噁心，它們並不像傷藥噴霧和櫻桃口味的咳嗽糖漿這麼受歡迎。

進化道具

寶可夢進化的運作方式相當神秘。有些寶可夢無須任何干預，只要成長茁壯並提升等級便會自然而然進化；有些需要透過養育並提升幸福度或忠誠度來進化；而也有些物種只能透過外在的影響才能進化，像是蘊藏在石頭的元素能量、或其他道具或機器。

寶可夢交換機器

訓練家們能夠交換彼此的寶可夢，但其過程並不只是交換彼此的精靈球這麼簡單，畢竟這樣並無法證明這個交換是雙方自願的。
為了解決這個問題，訓練家們需要將他們的寶可夢放到交換機器中，透過低輻射能量，將寶可夢從原本的精靈球轉移到另一個精靈球。而眾所周知，這種輻射能夠促使特定品種的寶可夢進化。

持有攜帶道具進化

有些寶可夢可能會對一些特殊的攜帶物品產生反應，像是工業金屬、充電器、和甚至糕點，這些物品在與交換機器的輻射波結合時將可能觸發某些寶可夢的進化反應。在野外，寶可夢也許終其一生都不會有機會接觸到這些人工製品，因此想要知道什麼物品能夠促使哪些寶可夢進化，就更為困難了。

你將會需要大量的耐心，努力研究，才能夠瞭解你需要什麼樣的道具來讓你的夥伴進化。
